# @gemini-mcp-rs/win32-arm64

Platform-specific binary package for **gemini-mcp-rs** (Windows arm64).

This package is not intended to be installed directly. Instead, install the main package:

```bash
npm install @missdeer/gemini-mcp-rs
```

Or run directly with npx:

```bash
npx @missdeer/gemini-mcp-rs
```

For more information, see the [main repository](https://github.com/missdeer/gemini-mcp-rs).
